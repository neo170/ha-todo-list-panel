# Todo List Panel

Modernes Todo-Listen Panel für Home Assistant mit Swipe-to-Delete und Checkbox-Unterstützung.

## Features

- 📋 Zeigt alle HA-Todo-Listen im Dropdown
- ✅ Todos abhaken (mit Durchstreichen)
- ➕ Neue Todos per Eingabefeld oder Enter hinzufügen
- 🗑️ Löschen per Button **oder Swipe nach links** (Touch)
- 📅 Fälligkeitsdatum wird angezeigt
- 🔄 Echtzeit-Sync mit HA State-Updates
- 🔢 Badge zeigt Anzahl offener Todos

## Voraussetzungen

- Home Assistant ≥ 2023.11
- Mindestens eine Todo-Liste in HA

## Installation via HACS

1. HACS öffnen → **Integrationen** → ⋮ → **Benutzerdefinierte Repositories**
2. URL eintragen: `https://github.com/YOUR_GITHUB_USER/ha-todo-list-panel`
3. Kategorie: **Integration**
4. **Hinzufügen** → Integration suchen → **Installieren**
5. HA neu starten
